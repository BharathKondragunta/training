Arrow function:
- it was introduced in ecmaScript6.version, and this version is introduved in the year 2015.
- it is used to reduce the number of lines in a code.
- there are two types of return keywords
    1. implicit return arrow function
        --> no need of using return keywords.
    eg:
    let a = () => console.log("hello");
    console.log(a);
    a()

    2. explict return arrow function
        --> return keyword is mandatory.
    eg:
    let b = () => {return "hello"}
    console.log(b);
    console.log(b());
