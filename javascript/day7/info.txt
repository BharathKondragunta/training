IIFE(intermediate invocking function expression) function:
    - it will be called immediately once the function object is created. 
    - it is called using the open and closed brackets ().
    - it should be terminated by using semi coloun(;).where the multiple IIFE functions are used 

syntax: (expression(function))()
eg:
    (function()
    {

    })()    

IIFE function with the arrow function:

(() => {

{)()

the named function converted into the IIFE function:

(function demo()
{

})()

objects in javascript:
- it is used to store the data in the form of key and value pairs.

eg:
let obj = {
    name: "Rahul",
    id : 1,
    designation: "Developer",
    arr : [10, 20, 30],
    boolean : true,
    bigint : 1n,

    {

    }
}

obj.demo();
